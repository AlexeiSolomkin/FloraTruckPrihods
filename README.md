_ДИСКЛЕЙМЕР: Этот проект навайбкожен за пару дней чисто для прикладных задача - автоматизировать часть моей работы.

Табель - это excel-документ, который содержит информацию о грузе, который везет машина.
Приход - это excel-документ, в котором выставляется счет клиентам по грузу, который едет в данной машине.
Данный сервис автоматизирует процесс создания Прихода.
